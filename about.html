An Information Retrieval system is a web application software which designed to parse existing lyrics files into corresponding tables which shows specifications of each scanned word.
This system includes several of functionalities that manipulates sql tables dynamically and manually as well.

In order to parse lyrics files and rearrange parsed words within corresponding tables, user must add specific files into the 'data' folder by clicking 'add' or by moving (cut and paste) an existing lyrics files which specially designed for this system.
*do not try to manually add files from another source.
This inverted files system aim to maintain real-time information on current lyrics files, so that the shown tables changes whenever each word is clicked or another available buttons. They do so by connecting to a database of words and hits upon sql queries actions.
